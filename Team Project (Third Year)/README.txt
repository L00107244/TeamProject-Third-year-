1. Start up mysql Workbench.
2. Create a new connection and call it hospital.
3. Click open a new script file and open simple DBv2 from the project folder.
4. Create a new schema and call it hospital.
5. Highlight all tables from simple DBv2 and double click on hospital schema and click the excute button.
6. Open up eclipse and point the workspace towards the project folder. (You may need to import the project in)
7. In the event of error right click on the project file and go to build path --> configure build path.
8. Go to libaries and if the jar file contains a red X then delete and add a new external jar file and navagate to project folder 
   and add the mysql Connecter jar file from the project folder.
